# Mechanics Contract

Project: "Медвежонок и таблица умножения"

These rules define the playable MVP mechanics. Design changes must not alter or weaken them.

1. Levels are loaded from `assets/data/levels.json`.
2. Each level contains 10 questions.
3. The map opens only available levels.
4. When the player reaches the mentor, a dialog must open.
5. After the dialog intro, a question must open.
6. Each question has 3 answer options.
7. A correct answer awards points.
8. A wrong answer shows a hint.
9. After 10 correct answers, the level is complete.
10. After a level is complete, the next level is unlocked.
11. After level 10, the final screen opens.
12. Design changes are not allowed to change these mechanics.
