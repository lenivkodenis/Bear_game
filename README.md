# Медвежонок и таблица умножения

Безопасный детский MVP мобильной 2D-игры на Flutter + Flame.

Белый медвежонок потерялся на севере и проходит локации, встречая мудрых северных животных. В следующих этапах мудрецы будут объяснять пользу таблицы умножения и задавать примеры.

## Что уже есть

- Главный экран с кнопками: "Начать игру", "Прогресс", "Родителям".
- Первый игровой экран на Flame.
- Простая 2D-сцена: снежный фон, платформа, медвежонок-placeholder и мудрец-placeholder.
- Управление влево/вправо и прыжок.
- Диалог-заглушка при подходе к мудрецу.
- Локальная модель прогресса через `shared_preferences`.
- Папки под будущие изображения, звук и данные.

## Запуск

В окружении с установленным Flutter:

```bash
flutter pub get
flutter run
```

Если платформа Android/iOS ещё не сгенерирована локально, выполните один раз:

```bash
flutter create . --platforms=android,ios
```

## Структура

- `lib/screens` — Flutter-экраны приложения.
- `lib/game` — Flame-игра и игровые компоненты.
- `lib/models` — модели данных.
- `lib/services` — сервисы, включая локальное хранение прогресса.
- `lib/widgets` — переиспользуемые виджеты интерфейса.
- `assets/data` — данные локаций.
- `assets/images` — будущая графика.
- `assets/audio` — будущие звуки и музыка.

## Принципы MVP

В проект не добавлены реклама, регистрация, онлайн-платежи, аналитика или внешние SDK для сбора данных.
